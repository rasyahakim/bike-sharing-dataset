

## Setup environment

```
conda create --name Dashboard-Main python=3.10
conda activate Dashboard-Main
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel
```

## Run steamlit app

```
streamlit run Fahri_Submission_Dashboard.py
```
